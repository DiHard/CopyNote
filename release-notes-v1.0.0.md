## CopyNote v1.0.0

Windows 11 tray utility for storing and quickly copying text snippets.

### Features
- One-click copy to clipboard
- Instant search across all entries
- Light / Dark / System theme
- English & Russian localization
- Import / Export backup (JSON)
- Autorun on Windows startup
- Frameless window with smooth slide animation
- Adaptive tray icon (auto-switches for light/dark taskbar)
- Single portable `.exe` (~7 MB, no installer needed)

### Requirements
- Windows 10 / 11
- WebView2 Runtime (pre-installed on Windows 11)

### Installation
Download `copynote.exe` below and run it. No installation required — the app starts minimized to the system tray. Click the tray icon to open.

### Data Storage
- Entries: `%APPDATA%\CopyNote\data.json`
- Settings: `%APPDATA%\CopyNote\settings.json`
- WebView2 cache: `%LOCALAPPDATA%\CopyNote\WebView2\`
